1. Uninstall unused programs.
2. Disable startup programs.
3. Clean up your hard drive.
4. Increase RAM if possible.

How do I change my Wi-Fi network name and password?

Access your router's settings by typing its IP address into a web browser (usually something like 192.168.1.1 or 192.168.0.1). Log in with your router’s admin credentials and go to the Wi-Fi settings section to change the network name (SSID) and password.

What can I do if my router keeps disconnecting from the internet?

Check for loose cables, update the router’s firmware, and make sure the router is not overheating. Interference from nearby electronic devices might also cause disconnections, so try relocating your router.

How do I set up a new Wi-Fi network at home?

Plug in your router to the modem and power supply. Connect a computer to the router using an Ethernet cable. Access the router's settings through a web browser using its IP address. Follow the setup wizard to create your Wi-Fi network name (SSID) and password.

How can I secure my Wi-Fi network?

1. Change the default login credentials of your router.
2. Use a strong, unique password for your Wi-Fi network.
3. Enable WPA3 or WPA2 encryption.
4. Disable remote access to your router.
5. Keep your router’s firmware up-to-date.
